Weather Station

This Weather Station measures temperature, pressure, and light levels using an ESP32 and displays live sensor data on a local website. Historical data is logged to the Ubidots platform, which also handles email notifications for sensor events.

I designed the full system architecture, connected sensors, coded the ESP32 logic in C++, set up web visualization using HTML/CSS, and configured the MQTT protocol with Ubidots.

ESP32 · BH1750 & BMP280 sensors · C++ · HTML/CSS · MQTT · PlatformIO · Ubidots · Visual Studio Code · Web server & APIs · IoT data handling.

A complete IoT system that collects environmental data and shares it both locally via an embedded web server and remotely through Ubidots. I also built email alert capabilities and a clean web interface showing real-time values.

  • How to use MQTT and send data from ESP32 to cloud platforms.
  • How to build real-time dashboards and integrate sensor data.
  • How to write modular C++ code for IoT projects and host web content on embedded systems.
  • Building perseverance and adaptability when facing hardware challenges.
  • Enhancing clarity and user focus in interface design for non-technical users.
